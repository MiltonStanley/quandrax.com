Convert your savegames to mods usable in the next game in the series. Download it here: Sourceforge page.

How do I use it?

Easy:

  1. Put your savegame in the same folder as the quandrax.exe file
  2. Either click the quandrax.exe file in your File Explorer, or run it from the command line with quandrax.exe
  3. Move the resulting mod/ folder and mySavegame.mod file into the mods folder of your game.
  4. Conquer the world.
